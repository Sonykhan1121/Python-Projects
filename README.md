## Project One: File Organizer

**Description**: 

This script organizes files in a specified directory by their extensions. Here's how it works:

- **Prompts for Directory Path**: The script starts by asking the user to enter the path of the directory they want to organize.
- **Lists Contents**: It then lists all files and directories within the specified path.
- **Identifies File Extensions**: For each file, the script determines its extension.
- **Checks for Existing Directories**:
  - If a directory named after the extension already exists, the script moves the file into that directory.
  - If such a directory does not exist, the script creates it.
- **Moves Files**: Once the appropriate directory is confirmed or created, the script moves the file into the corresponding directory.

This process helps in organizing files neatly by their extensions, making the directory more manageable.

**How it works**:
1. **Step 1**: Enter the path.
2. **Step 2**: Press enter.
3. **Step 3**: Check your path in your computer.

## Project Two: QR Code Generator

**Description**: 

This project generates QR codes using the `qrcode` library in Python. There are two parts to this project: a simple QR code generator and a customizable QR code generator.

### Simple QR Code Generator

This part of the project generates a basic QR code that encodes a URL (e.g., your GitHub profile).

- **Data to Encode**: A URL or any other string data.
- **Output**: A PNG image file of the QR code.

### Customizable QR Code Generator

In the customizable QR code generator, you can set various parameters:

- **Version**: Determines the size of the QR code.
- **Error Correction**: Levels of error correction available (L, M, Q, H).
- **Box Size**: Size of each box in pixels.
- **Border**: Width of the border (minimum is 4).
- **Colors**: Customizable fill and background colors.

### How it works

1. **Install Required Libraries**:
    - Ensure you have the `qrcode` and `Pillow` libraries installed.
2. **Create QR Code Object**:
    - Configure the QR code with the desired parameters.
3. **Add Data to QR Code**:
    - Add the data you want to encode (e.g., URL).
4. **Generate and Save the QR Code**:
    - Generate the QR code and save it as an image file.
